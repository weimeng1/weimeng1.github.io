基于hexo博客上的matery主题美化以及优化

matery主题优化
发布日期:   2020-04-30

前言

1.上一篇教你们搭建了一个属于自己的博客并且部署到了coding上，但是有个问题他那个博客不好看怎么办我们就可以去下载一个主题我推荐的就是这个matery主题，因为我用的就是这个总体感觉还不错。

安装matery主题

首先我们打开储存博客的文件夹里面会有个themes命名的文件夹打开右键运行Git Bash Here输入这一段代码 

git clone https://github.com/blinkfox/hexo-theme-matery.git

他就会自动生成一个hexo-theme-matery的文件夹然后我们先来到hexoblog（就是你博客的文件夹）打开之后打开命名为_config.yml的文件_config.yml是你hexo博客的配置文件有些地方是可以改的我们用 VScode打开然后到翻到底最后几行里有一段这个代码

这个代码差不多在95行左右的样子然后我们吧那个theme：改成你在themes文件夹里生成的主题文件的名字我这个图片是改好之后的样子改好之后，我们保存然后到我们hexoblog再打开Git Bash Here然后运行hexo clean（说明一下运行这段代码之后他会删除你hexoblog文件夹下的public文件）然后我们再运行hexo g（运行这段代码就会把你这些改好了的文件和代码重新做一个public文件夹PS：我也不知道这样讲对不对）然后我们再运行hexo d（运行这段会把你本地的文件数据部署到coding）这时候打开你的博客大概的布局就是这样的

当然你的肯定和我的不一样我的是改好了的接下来我们就来自定义你的主题。

自定义主题

我们打开hexoblog的_config.yml然后就是开始那一段就是我们需要改的
第一段

# Site
title: 罗 # 这里可以改左上角的字
subtitle: さあ # 这里是改中间的文字
description: ''
keywords:
author: 剑仙 #这里是改你的昵称
language: zh-CN # 这个你就照着我这个改这个是用来显示语言的
timezone: ''

我给大家搞个图吧title: 罗 # 这里可以改左上角的字这个对应的区域是logo 旁边的文字

这一段subtitle: さあ # 这里是改中间的文字是中间的文字

这一段author: 剑仙 #这里是改你的昵称是改你的昵称

到这里的话博客的配置就差不多了。

新建分类 categories 页

categories 页是用来展示所有分类的页面，如果 source 目录下还没有 categories/index.md 文件，那么就需要新建一个，打开git输入：

hexo new page "categories"

然后他就会到你的这个路径生成一个文件/source/categories/index.md
把他们替换变成这样 

---
title: categories
date: 2018-09-30 17:25:30
type: "categories"
layout: "categories"
---

提示一下：下面所新建页面都会在hexoblog\source下面生成文件

新建标签 tags 页

tags 页是用来展示所有标签的页面，如果 source 目录下还没有 tags/index.md 文件，那么就需要新建一个据需输入命令： 

hexo new page "tags"

然后就会在我上面说的那个路径生成一个tag文件夹打开然后再打开里面的index.md把这些代码替换上去 

---
title: tags
date: 2018-09-30 18:23:38
type: "tags"
layout: "tags"
---

新建关于我 about 页

路径就不说了直接输入命令hexo new page "about"然后打开about文件夹里的文件换这个代码

---
title: about
date: 2018-09-30 17:25:30
type: "about"
layout: "about"
---

新建友情连接 friends 页

命令：hexo new page "friends"然后打开friends文件夹里的文件替换成这个： 

---
title: friends
date: 2018-12-12 21:25:30
type: "friends"
layout: "friends"
---

代码高亮

由于 Hexo 自带的代码高亮主题显示不好看，所以主题中使用到了 hexo-prism-plugin 的 Hexo 插件来做代码高亮，安装命令如下： 

npm i -S hexo-prism-plugin

然后，修改 Hexo 根目录下 _config.yml 文件中 highlight.enable 的值为 false，并新增 prism 插件相关的配置，主要配置如下： 

highlight:
  enable: false

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'tomorrow'
  line_number: false    # default false
  custom_css:

搜索

本主题中还使用到了 hexo-generator-search 的 Hexo 插件来做内容搜索，安装命令如下：

npm install hexo-generator-search --save

在 Hexo 根目录下的 _config.yml 文件中，新增以下的配置项： 

search:
  path: search.xml
  field: post

修改主题的配置文件

主题配置文件位置在hexoblog\themes\hexo-theme-matery\_config.yml打开里面都有说明的有的地方是默认关闭的你可以看到上面的代码flase就是关闭true就是打开里面主要可以修改的就是

  • 我的梦想

  • 音乐配置（默认关闭你打开就行如果不会的话看我下面的代码）

    # Whether to display the musics.
# 是否在首页显示音乐.
music:
enable: true #默认是flase你改为true
title: #非吸底模式有效
  enable: true
  show: 听听音乐
autoHide: true    # hide automaticaly
server: netease   #require    music platform: netease, tencent, kugou, xiami, baidu 这里是音乐源
type: playlist    #require song, playlist, album, search, artist
id: 4939093738    #require    song id / playlist id / album id / search keyword 音乐歌单的id
fixed: true       # 开启吸底模式
autoplay: true   # 是否自动播放
theme: '#42b983'
loop: 'all'       # 音频循环播放, 可选值: 'all', 'one', 'none'
order: 'random'   # 音频循环顺序, 可选值: 'list', 'random'
preload: 'auto'   # 预加载，可选值: 'none', 'metadata', 'auto'
volume: 0.7       # 默认音量，请注意播放器会记忆用户设置，用户手动设置音量后默认音量即失效
listFolded: true  # 列表默认折叠
    然后呢我考虑到有的可能不知道自己音乐歌单的ID是啥我教你们一个办法你们打开浏览器进网易云音乐然后点击歌单就会这样箭头指出的地方就是id
    接下来的我就不一一说名怎么改可以自行百度都很容易

  • 个人信息

  • 我的技能

  • 我的项目（可修改可关闭）

  • 我的相册（默认是3张图片如果需要添加图片按着格式往下面加图片的路径就行）

  • 我的技能（可以改技能是什么还有技能的百分比完成度）

    修改图片

    图片这个我也不过多赘述你看看你博客的图片到文件夹，里面找到对应的把名字复制下来把要替换的图片改成你要替换的图片名字就行这个是图片的路径路径：
    hexoblog\themes\hexo-theme-matery\source 里面有几个我们要改的首先就是一个网页的logo：favicon.png你们把你们要改的替换这个就行建议logo用ps修改尺寸最好相同
    然后改完之后我们进到那个路径下的medias文件夹我一个一个和你们介绍是改哪的
  • banner文件夹
    这是你进去博客首页的图片一共7张每天换一张
  • featureimages文件夹
    这相册的文件夹
  • reward文件夹
    这是你打赏的支付宝和微信二维码

    这些改完之后就是些散图了

  • avatar.jpg
    这个图片改的是你的头像
  • logo.png
    这个图片是改你首页左上角的一个logo
